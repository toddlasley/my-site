ovrload

A simple web app used for tracking weight or resistance based training exercises.
Built front to back with TypeScript, running Angular on the front-end and Node.js and Express.js on the back. Both the front-end and back-end applications run in their own isolated containers which are hosted via DigitalOcean Droplets. The data storage is implemented with a MySQL database which is hosted via an Amazon RDS instance.
